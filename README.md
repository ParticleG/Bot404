# BOT404

A auto video downloader plugin for bot based on CQ and youtube-dlc.

## Requirements

```shell
pip install -r requirements.txt
```

## Start API-backend

Runtime(Production) environment:
```
uvicorn <class-name>:app
```

Or hot-edit:

e.g.
```
uvicorn web-interface-demo:app --reload
```
